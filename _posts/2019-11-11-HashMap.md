# Summary
```java
public class HashMap<K,V> extends AbstractMap<K,V>
        implements Map<K,V>, Cloneable, Serializable
```
基于哈希表实现的映射接口。此实现提供所有可选映射操作，并允许空值和空键。(HashMap类大致相当于Hashtable，只是它是不同步的，并且允许为空。)该类不保证映射的顺序;特别是，它不能保证顺序会随着时间保持不变。

这个实现为基本操作(get和put)提供了稳定的时间性能，假设hash函数将元素恰当地分散到各个桶中。集合视图的迭代需要与HashMap实例的“容量”(桶的数量)及其大小(键值映射的数量)成比例的时间。因此，如果迭代性能很重要，那么不要将初始容量设置得太高(或负载因子过低)。

HashMap实例有两个影响其性能的参数:初始容量和负载因子。容量是哈希表中的桶数，初始容量就是创建哈希表时的容量。负载因子是衡量在哈希表的容量被自动增加之前，哈希表被允许获得多少满的度量。当哈希表中的条目数超过负载因子和当前容量的乘积时，哈希表将被重新哈希(即重新构建内部数据结构)，这样哈希表大约是桶数的两倍。

一般来说，默认的负载因子(.75)在时间和空间成本之间提供了很好的权衡。较高的值减少了空间开销，但增加了查找成本(反映在HashMap类的大多数操作中，包括get和put)。在设置map的初始容量时，应该考虑map中预期条目的数量及其负载因子，从而最小化rehash操作的数量。如果初始容量大于最大条目数除以负载因子，则不会发生任何重哈希操作。

如果要将许多映射存储在HashMap实例中，那么使用足够大的容量创建映射将比让映射根据需要执行自动哈希来增长表更有效地存储映射。注意，使用具有相同hashCode()的多个键肯定会降低任何散列表的性能。为了改善影响，当键是可比较的时，该类可以使用键之间的比较顺序来帮助断开连接。

注意，这个实现不是同步的。如果多个线程同时访问一个散列映射，并且其中至少有一个线程从结构上修改了映射，那么它必须在外部同步。(结构修改是添加或删除一个或多个映射的任何操作;仅更改与实例已经包含的键关联的值不是结构修改。)这通常是通过在一些自然封装映射的对象上进行同步来实现的。如果不存在这样的对象，则应该使用集合“包装”Collection.synchronizedMap方法。这最好在创建时完成，以防止意外的不同步访问map:

```java
Map m = Collections.synchronizedMap(new HashMap(...));
```

这个类的所有“集合视图方法”返回的迭代器都是快速失败的:如果在创建迭代器之后的任何时候对映射进行结构上的修改，除了通过迭代器自己的remove方法之外，迭代器将抛出ConcurrentModificationException。因此，在面对并发修改时，迭代器会快速而干净地失败，而不是在将来某个不确定的时间冒着任意的、不确定的行为的风险。

注意，不能保证迭代器的快速故障行为，因为通常来说，在存在非同步并发修改的情况下，不可能做出任何严格的保证。故障快速迭代器以最大的努力抛出ConcurrentModificationException。因此，编写一个依赖于此异常来判断其正确性的程序是错误的:迭代器的快速故障行为应该只用于检测bug。

# Nested Class
## Node
```java
static class Node<K,V> implements Map.Entry<K,V> {
// 比Entry多个next和hash
    final int hash;
    final K key;
    V value;
    Node<K,V> next;

    Node(int hash, K key, V value, Node<K,V> next) {
        this.hash = hash;
        this.key = key;
        this.value = value;
        this.next = next;
    }

    public final K getKey()        {
        return key;
    }

    public final V getValue()      {
        return value;
    }

    public final String toString() {
        return key + "=" + value;
    }

    public final int hashCode() {
        return Objects.hashCode(key) ^ Objects.hashCode(value);
    }

    public final V setValue(V newValue) {
        V oldValue = value;
        value = newValue;
        return oldValue;
    }

    public final boolean equals(Object o) {
        if (o == this)
            return true;
        if (o instanceof Map.Entry) {
            Map.Entry<?,?> e = (Map.Entry<?,?>)o;
            if (Objects.equals(key, e.getKey()) &&
                Objects.equals(value, e.getValue()))
                return true;
        }
        return false;
    }
}


```






# Method
**Static utilities**
## hash
```java
static final int hash(Object key) {
    int h;
    return (key == null) ? 0 : (h = key.hashCode()) ^ (h >>> 16);
}
```
将高于16位的影响向下传播，避免低位hash出现冲突发生冲突。

桶数一般都是低于16位的，所以在 & 桶数会产生较大冲突，这样可以使低位变化大些。

在&桶数-1后，比如7 =》 111 与操作，高位全部归零，只有低三位影响结果，容易发生冲突，所以做一个扰动函数比较好。















## comparableClassFor
```java
static Class<?> comparableClassFor(Object x) {
    if (x instanceof Comparable) {
        Class<?> c; Type[] ts, as; Type t; ParameterizedType p;
        if ((c = x.getClass()) == String.class) // bypass checks
            return c;
        if ((ts = c.getGenericInterfaces()) != null) {
            for (int i = 0; i < ts.length; ++i) {
            if (((t = ts[i]) instanceof ParameterizedType) &&
                ((p = (ParameterizedType)t).getRawType() ==
                Comparable.class) &&
                (as = p.getActualTypeArguments()) != null &&
                as.length == 1 && as[0] == c) // type arg is c
                return c;
            }
        }
    }
    return null;
}
```

## tableSizeFor
```java
static final int tableSizeFor(int cap) {
    int n = cap - 1;
    n |= n >>> 1;
    n |= n >>> 2;
    n |= n >>> 4;
    n |= n >>> 8;
    n |= n >>> 16;
    return (n < 0) ? 1 : (n >= MAXIMUM_CAPACITY) ? MAXIMUM_CAPACITY : n + 1;
}
```
返回给定目标容量的两次幂大小。

不管你传啥玩意，都能给你转成2的幂，牛逼

原理就是把低位开始每个位都变成1，结果加1，就会变成2的次幂。

```java
final void putMapEntries(Map<? extends K, ? extends V> m, boolean evict) {
    int s = m.size();
    if (s > 0) {
    // 初始化桶大小
        if (table == null) {
            // pre-sizev 桶数目吧
            float ft = ((float)s / loadFactor) + 1.0F;
            int t = ((ft < (float)MAXIMUM_CAPACITY) ?
                        (int)ft : MAXIMUM_CAPACITY);

        else if (s > threshold)

        for (Map.Entry<? extends K, ? extends V> e : m.entrySet()) {
            K key = e.getKey();
            V value = e.getValue();
            putVal(hash(key), key, value, false, evict);
        }
    }
}
```

## getNode
```java
final Node<K,V> getNode(int hash, Object key) {
    Node<K,V>[] tab; Node<K,V> first, e; int n; K k;
    // 判断桶数目不为空
    if ((tab = table) != null && (n = tab.length) > 0 &&
    // 找桶，hash & 桶的大小-1得桶的位置
            (first = tab[(n - 1) & hash]) != null) {
            
        if (first.hash == hash && // always check first node 检查第一个值
            ((k = first.key) == key || (key != null && key.equals(k))))
            return first;
        if ((e = first.next) != null) {
        // 这个是处理树结构
            if (first instanceof TreeNode)
                return ((TreeNode<K,V>)first).getTreeNode(hash, key);
            // hash相等说明在一个桶里，再比较key相等的话，就找到了
            // 这个应该就是链表了
            do {
                if (e.hash == hash &&
                    ((k = e.key) == key || (key != null && key.equals(k))))
                    return e;
            } while ((e = e.next) != null);
            
        }
    }
    return null;
}
```

## putVal
```java
final V putVal(int hash, K key, V value, boolean onlyIfAbsent,
                boolean evict) {
    // 桶[]
    Node<K,V>[] tab; 
    // 一个桶
    Node<K,V> p; 
    // n:桶数量；i:定位到的桶的位置
    int n, i;
    // 如果桶[]为null，或者长度为0，则初始化桶[]
    if ((tab = table) == null || (n = tab.length) == 0)
        n = (tab = resize()).length;
    // 找到当前键值所在的桶，如果不存在，则初始化一个桶
    if ((p = tab[i = (n - 1) & hash]) == null)
        tab[i] = newNode(hash, key, value, null);
    else { // 如果找到了桶，则寻找节点
        Node<K,V> e; K k;
        // 如果第一个节点就是，那么取出
        if (p.hash == hash &&
            ((k = p.key) == key || (key != null && key.equals(k))))
            e = p;
        // 看是否是树节点
        else if (p instanceof TreeNode)
            e = ((TreeNode<K,V>)p).putTreeVal(this, tab, hash, key, value);
        else { // 处理链表
            // 无限循环
            for (int binCount = 0; ; ++binCount) {
                // 终止条件 新元素
                if ((e = p.next) == null) {
                    p.next = newNode(hash, key, value, null);
                    // 元素多，则变为树结构
                    if (binCount >= TREEIFY_THRESHOLD - 1) // -1 for 1st
                        treeifyBin(tab, hash);
                    break;
                }
                
                // 终止条件 已存在key
                if (e.hash == hash &&
                    ((k = e.key) == key || (key != null && key.equals(k))))
                    break;
                p = e;
            }
        }
        // 重复key的话，需要取出旧值
        if (e != null) { // existing mapping for key
            V oldValue = e.value;
            if (!onlyIfAbsent || oldValue == null)
                e.value = value;
            // LinkHashMap实现此方法
            afterNodeAccess(e);
            return oldValue;
        }
    }
    ++modCount; // 修改了
    // 如果数量大于阈值，扩容
    if (++size > threshold)
        resize();
    // LinkHashMap实现此方法
    afterNodeInsertion(evict);
    return null;
}
```

## resize
```java
    final Node<K,V>[] resize() {
        // 旧的桶[]
        Node<K,V>[] oldTab = table;
        // 旧的数
        int oldCap = (oldTab == null) ? 0 : oldTab.length;
        // 旧的元素阈值
        int oldThr = threshold;
        // 新的桶数，新的元素阈值
        int newCap, newThr = 0;
        
        // 如果桶数大于0
        if (oldCap  > 0) {
        // 旧桶数大于最大桶容量的话，阈值变为最大int值
            if (oldCap >= MAXIMUM_CAPACITY) {
                threshold = Integer.MAX_VALUE;
                return oldTab;
            }
            // 新桶数变为旧桶数的二倍，并且小于最大桶容量，旧桶数大于默认桶容量的话，新阈值是就阈值的二倍
            else if ((newCap = oldCap << 1) < MAXIMUM_CAPACITY &&
                     oldCap >= DEFAULT_INITIAL_CAPACITY)
                // 新元素阈值等于旧的元素阈值的二倍
                newThr = oldThr << 1; // double threshold 双倍阈值
        }
        
        // 如果实际桶数为0，旧理论桶数大于0
        // 这里是指初始化桶的时候会把桶数赋值给threshold，这里取出来用
        else if (oldThr > 0) // initial capacity was placed in threshold
            // 这里就用到初始化的newCap来替换threshold
            newCap = oldThr;
        else {               // zero initial threshold signifies using defaults
        // 如果都为0，新桶数为默认，16
            newCap = DEFAULT_INITIAL_CAPACITY;
            // 新的元素容量（桶数 * 负载因子）
            newThr = (int)(DEFAULT_LOAD_FACTOR * DEFAULT_INITIAL_CAPACITY);
        }
        
        
        if (newThr == 0) { // 旧的Thr赋值给newCap了，这里从新赋值
            float ft = (float)newCap * loadFactor;
            newThr = (newCap < MAXIMUM_CAPACITY && ft < (float)MAXIMUM_CAPACITY ?
                      (int)ft : Integer.MAX_VALUE);
        }
        
        // 阈值赋值结束，下面开始从新分配
        threshold = newThr;
        @SuppressWarnings({"rawtypes","unchecked"})
            Node<K,V>[] newTab = (Node<K,V>[])new Node[newCap];
        table = newTab;
        
        // 如果之前有元素的话，这里需要从新分配元素
        if (oldTab != null) {
        //  循环桶数组，取元素
            for (int j = 0; j < oldCap; ++j) {
                Node<K,V> e;
                // 如果桶里有元素的话
                if ((e = oldTab[j]) != null) {
                // 清空引用
                    oldTab[j] = null;
                    if (e.next == null)
                    // 如果只有一个元素的话，分配到新桶
                        newTab[e.hash & (newCap - 1)] = e;
                    else if (e instanceof TreeNode)
                    // 如果是树的话
                        ((TreeNode<K,V>)e).split(this, newTab, j, oldCap);
                    else { // preserve order
                    // 链表的话
                        // 该链表留在原位置
                        Node<K,V> loHead = null, loTail = null;
                        // 该链表索引加上oldCap
                        Node<K,V> hiHead = null, hiTail = null;
                        Node<K,V> next;
                        do {
                            next = e.next;
                            // 假如oldCap为1000，那么扩容后的为10000，-1就等于1111，
                            // 所以只会等于0或oldCap,0就在原位置，oldCap在索引加oldCap位置
                            if ((e.hash & oldCap) == 0) {
                                if (loTail == null)
                                    loHead = e;
                                else
                                    loTail.next = e;
                                loTail = e;
                            }
                            else {
                                if (hiTail == null)
                                    hiHead = e;
                                else
                                    hiTail.next = e;
                                hiTail = e;
                            }
                        } while ((e = next) != null);
                        
                        if (loTail != null) {
                            loTail.next = null;
                            newTab[j] = loHead;
                        }
                        if (hiTail != null) {
                            hiTail.next = null;
                            newTab[j + oldCap] = hiHead;
                        }
                    }
                }
            }
        }
```

## removeNode
```java
    final Node<K,V> removeNode(int hash, Object key, Object value,
                               boolean matchValue, boolean movable) {
        Node<K,V>[] tab; Node<K,V> p; int n, index;
        if ((tab = table) != null && (n = tab.length) > 0 &&
            (p = tab[index = (n - 1) & hash]) != null) {
            // 找到元素存在的桶
            Node<K,V> node = null, e; K k; V v;
            if (p.hash == hash &&
                ((k = p.key) == key || (key != null && key.equals(k))))
                // 如果第一个节点就是
                node = p;
            else if ((e = p.next) != null) {
                if (p instanceof TreeNode) // 若为树，则在树中查找
                    node = ((TreeNode<K,V>)p).getTreeNode(hash, key);
                else { // 链表的话，循环查找
                    do {
                        if (e.hash == hash &&
                            ((k = e.key) == key ||
                             (key != null && key.equals(k)))) {
                             // 找到后退出循环
                            node = e;
                            break;
                        }
                        // p为e的前一个值
                        p = e;
                    } while ((e = e.next) != null);
                }
            }
            // 如果找到节点，或matchValue为null，或matchValue为true但值相等的话
            if (node != null && (!matchValue || (v = node.value) == value ||
                                 (value != null && value.equals(v)))) {
                // 如果为树结构，则在树中删除节点
                if (node instanceof TreeNode)
                    ((TreeNode<K,V>)node).removeTreeNode(this, tab, movable);
                else if (node == p) // 如果第一个节点就是，而且为链表结构
                    tab[index] = node.next;
                else // 如果非第一个节点的链表
                    p.next = node.next;
                ++modCount; // 结构修改
                --size; // 大小减一
                afterNodeRemoval(node);
                return node;
            }
        }
        return null;
    }
```

```java
    public V computeIfAbsent(K key,

            throw new NullPointerException();
```

```java
    @Override
    public V merge(K key, V value,
                   BiFunction<? super V, ? super V, ? extends V> remappingFunction) {
        if (value == null)
            throw new NullPointerException();
        if (remappingFunction == null)
            throw new NullPointerException();
        int hash = hash(key);
        Node<K,V>[] tab; Node<K,V> first; int n, i;
        int binCount = 0;
        TreeNode<K,V> t = null;
        Node<K,V> old = null;
        //  如果size大于容量，或表为空，则扩容
        if (size > threshold || (tab = table) == null ||
            (n = tab.length) == 0)
            n = (tab = resize()).length;
         // 找到key的桶   
        if ((first = tab[i = (n - 1) & hash]) != null) {
        // 如果是树，拿到节点
            if (first instanceof TreeNode)
                old = (t = (TreeNode<K,V>)first).getTreeNode(hash, key);
            else {
            // 链表的话，循环找到节点
                Node<K,V> e = first; K k;
                do {
                    if (e.hash == hash &&
                        ((k = e.key) == key || (key != null && key.equals(k)))) {
                        old = e;
                        break;
                    }
                     // 拿到节点的长度
                    ++binCount;
                } while ((e = e.next) != null);
            }
        }
        // 旧节点不为null的话
        if (old != null) {
            V v;
            if (old.value != null)
                v = remappingFunction.apply(old.value, value);
            else
                v = value;
                // 值不为null的话，替换
            if (v != null) {
                old.value = v;
                afterNodeAccess(old);
            }
            else // 值为null，删除
                removeNode(hash, key, null, false, true);
            return v;
        }
        if (value != null) {// 无旧节点，插入新节点
            if (t != null) // 树的情况
                t.putTreeVal(this, tab, hash, key, value);
            else { // 链表的情况

                if (binCount >= TREEIFY_THRESHOLD - 1) // 大于8变成树
                    treeifyBin(tab, hash);
            }
            ++modCount;
            ++size;
            afterNodeInsertion(true);
        }
        return value;
    }
```

```java
    private void readObject(java.io.ObjectInputStream s)
        throws IOException, ClassNotFoundException {
        // Read in the threshold (ignored), loadfactor, and any hidden stuff
        s.defaultReadObject();
        reinitialize();
        if (loadFactor <= 0 || Float.isNaN(loadFactor))
            throw new InvalidObjectException("Illegal load factor: " +
                                             loadFactor);
        s.readInt();                // Read and ignore number of buckets
        int mappings = s.readInt(); // Read number of mappings (size)
        if (mappings < 0)
            throw new InvalidObjectException("Illegal mappings count: " +
                                             mappings);
        else if (mappings > 0) { // (if zero, use defaults)
            // Size the table using given load factor only if within
            // range of 0.25...4.0
            float lf = Math.min(Math.max(0.25f, loadFactor), 4.0f);
            float fc = (float)mappings / lf + 1.0f;
            int cap = ((fc < DEFAULT_INITIAL_CAPACITY) ?
                       DEFAULT_INITIAL_CAPACITY :
                       (fc >= MAXIMUM_CAPACITY) ?
                       MAXIMUM_CAPACITY :
                       tableSizeFor((int)fc));
            float ft = (float)cap * lf;
            threshold = ((cap < MAXIMUM_CAPACITY && ft < MAXIMUM_CAPACITY) ?
                         (int)ft : Integer.MAX_VALUE);

            // Check Map.Entry[].class since it's the nearest public type to
            // what we're actually creating.
            SharedSecrets.getJavaOISAccess().checkArray(s, Map.Entry[].class, cap);
            @SuppressWarnings({"rawtypes","unchecked"})
            Node<K,V>[] tab = (Node<K,V>[])new Node[cap];
            table = tab;

            // Read the keys and values, and put the mappings in the HashMap
            for (int i = 0; i < mappings; i++) {
                @SuppressWarnings("unchecked")
                    K key = (K) s.readObject();
                @SuppressWarnings("unchecked")
                    V value = (V) s.readObject();
                putVal(hash(key), key, value, false, false);
            }
        }
```

## HashIterator
```java
    abstract class HashIterator {
        Node<K,V> next;        // 返回的下个entry
        Node<K,V> current;     // 当前 entry
        int expectedModCount;  // 为了 快速失败
        int index;             // 当前位置

        HashIterator() { // 构造方法
            expectedModCount = modCount;
            Node<K,V>[] t = table;
            current = next = null;
            index = 0;
            if (t != null && size > 0) { // 进入第一个 非null桶的entry
                do {} while (index < t.length && (next = t[index++]) == null);
            }
        }

// 是否有下一个元素
        public final boolean hasNext() {
            return next != null;
        }

// 返回下一个节点
        final Node<K,V> nextNode() {
            Node<K,V>[] t;
            Node<K,V> e = next;
            if (modCount != expectedModCount) // 防止并发修改
                throw new ConcurrentModificationException();
            if (e == null) // 如果当前为null
                throw new NoSuchElementException();
            if ((next = (current = e).next) == null && (t = table) != null) { // 若下一个节点为null，表不为null，拿到下一个不为null桶的第一个节点
                do {} while (index < t.length && (next = t[index++]) == null);
            }
            return e;
        }

// 移除节点
        public final void remove() {
            Node<K,V> p = current;
            if (p == null)
                throw new IllegalStateException();
            if (modCount != expectedModCount)
                throw new ConcurrentModificationException();
            current = null;
            // 移除之后当前值为null，也就是说你不能移除两次当前值。
            K key = p.key;
            removeNode(hash(key), key, null, false, false);
            expectedModCount = modCount;
        }
    }
```

```java
    static class HashMapSpliterator<K,V> {
        final HashMap<K,V> map;
        Node<K,V> current;          // current node
        int index;                  // current index, modified on advance/split
        int fence;                  // one past last index
        int est;                    // size estimate

                           int expectedModCount) {

                hi = fence = (tab == null) ? 0 : tab.length;

            return hi;
        }

            getFence(); // force init
            return (long) est;
```

## TreeNode
```java
    static final class TreeNode<K,V> extends LinkedHashMap.Entry<K,V> {
        TreeNode<K,V> parent;  // 红黑树
        TreeNode<K,V> left;
        TreeNode<K,V> right;
        TreeNode<K,V> prev;    // needed to unlink next upon deletion
        boolean red;
        TreeNode(int hash, K key, V val, Node<K,V> next) {
            super(hash, key, val, next);
        }

        /**
         * 返回根节点
         */
        final TreeNode<K,V> root() {
            for (TreeNode<K,V> r = this, p;;) {
                if ((p = r.parent) == null)
                    return r;
                r = p;
            }
        }

        /**
         * Ensures that the given root is the first node of its bin.
         * 把root节点提到前面，链表变为树后
         */
        static <K,V> void moveRootToFront(Node<K,V>[] tab, TreeNode<K,V> root) {
            int n;
            if (root != null && tab != null && (n = tab.length) > 0) {
                int index = (n - 1) & root.hash;
                TreeNode<K,V> first = (TreeNode<K,V>)tab[index];
                if (root != first) {
                    Node<K,V> rn;
                    tab[index] = root;
                    TreeNode<K,V> rp = root.prev;
                    if ((rn = root.next) != null)
                        ((TreeNode<K,V>)rn).prev = rp;
                    if (rp != null)
                        rp.next = rn;
                    if (first != null)
                        first.prev = root;
                    root.next = first;
                    root.prev = null;
                }
                assert checkInvariants(root);
            }
        }

        /**
         * 以给定的hash和key从root开始找节点
         * kc 是 键的比较器
         */
        final TreeNode<K,V> find(int h, Object k, Class<?> kc) {
            TreeNode<K,V> p = this;
            // 循环找
            do {
                int ph, dir; K pk; 
                TreeNode<K,V> pl = p.left, pr = p.right, q;
                if ((ph = p.hash) > h) // 如果p的hash大于要找的key的hash，去左边找
                    p = pl;
                else if (ph < h) // 如果p的hash小于于要找的key的hash，去右边找
                    p = pr;
                else if ((pk = p.key) == k || (k != null && k.equals(pk))) // 找到了直接返回
                    return p;
                else if (pl == null) // 从这里开始hash判断不了位置了，只能盲找
                    p = pr;
                else if (pr == null)
                    p = pl;
                else if ((kc != null ||
                          (kc = comparableClassFor(k)) != null) && // 键的运行时类型
                         (dir = compareComparables(kc, k, pk)) != 0) // 比较器找
                    p = (dir < 0) ? pl : pr;
                else if ((q = pr.find(h, k, kc)) != null) // 比较器也找不到，只能递归左边
                    return q;
                else // 再找右边
                    p = pl;
            } while (p != null);
            return null;
        }

        /**
         * 从根节点开始找值，比较器为null
         */
        final TreeNode<K,V> getTreeNode(int h, Object k) {
            return ((parent != null) ? root() : this).find(h, k, null);
        }

        /**
         * hash相等的话，比较Object的hashcode大小
         */
        static int tieBreakOrder(Object a, Object b) {
            int d;
            if (a == null || b == null ||
                (d = a.getClass().getName().
                 compareTo(b.getClass().getName())) == 0)
                d = (System.identityHashCode(a) <= System.identityHashCode(b) ?
                     -1 : 1);
            return d;
        }

        /**
         * 树链表变为树结构
         * @return root of tree
         */
        final void treeify(Node<K,V>[] tab) {
            TreeNode<K,V> root = null;
            for (TreeNode<K,V> x = this, next; x != null; x = next) {
                next = (TreeNode<K,V>)x.next;
                // 让x左右都等于null
                x.left = x.right = null;
                // 初始化根节点，颜色为黑色，父节点为null，赋值x
                if (root == null) {
                    x.parent = null;
                    x.red = false;
                    root = x;
                }
                else {
                    K k = x.key; // 键
                    int h = x.hash; // 键的hash
                    Class<?> kc = null; // 键的运行时类型
                    for (TreeNode<K,V> p = root;;) { // 无限循环，从root开始
                    // dir为比较值，ph为hash
                        int dir, ph;
                        // pk为键的key
                        K pk = p.key;
                        if ((ph = p.hash) > h) // x在左边
                            dir = -1;
                        else if (ph < h) // x在右边
                            dir = 1;
                        else if ((kc == null &&
                                  (kc = comparableClassFor(k)) == null) ||
                                 (dir = compareComparables(kc, k, pk)) == 0)
                                 // hash相等，只能比较键的比较器来比较
                                 // 比较器也无法比较，那谁知道这人的equals咋写的，只能比较调用Object的hashcode方法了
                            dir = tieBreakOrder(k, pk);

                        TreeNode<K,V> xp = p;
                        if ((p = (dir <= 0) ? p.left : p.right) == null) {
                        // 如果遇到空位，就插入
                            x.parent = xp;
                            if (dir <= 0)
                                xp.left = x;
                            else
                                xp.right = x;
                                // 这里平衡一下树
                            root = balanceInsertion(root, x);
                            break;
                        }
                    }
                }
            }
            moveRootToFront(tab, root);
        }

        /**
         * 树退化为链表
         */
        final Node<K,V> untreeify(HashMap<K,V> map) {
            Node<K,V> hd = null, tl = null;
            for (Node<K,V> q = this; q != null; q = q.next) {
                Node<K,V> p = map.replacementNode(q, null);
                if (tl == null)
                    hd = p;
                else
                    tl.next = p;
                tl = p;
            }
            return hd;
        }

        /**
         * 红黑树put值
         */
        final TreeNode<K,V> putTreeVal(HashMap<K,V> map, Node<K,V>[] tab,
                                       int h, K k, V v) {
            Class<?> kc = null; // 可比较的键的运行时类型
            boolean searched = false; // 确保只找一次
            TreeNode<K,V> root = (parent != null) ? root() : this; // 根节点
            for (TreeNode<K,V> p = root;;) {
                int dir, ph; K pk;
                if ((ph = p.hash) > h) // 大于放左边
                    dir = -1;
                else if (ph < h) // 小于放右边
                    dir = 1;
                else if ((pk = p.key) == k || (k != null && k.equals(pk))) // == 或 equals相等就取出
                    return p;
                else if ((kc == null &&
                          (kc = comparableClassFor(k)) == null) ||
                         (dir = compareComparables(kc, k, pk)) == 0) {
                         // 这是hash相等，但是equals不等，还没有比较器火比较完等于0的情况，盲查，查到了就返回
                    if (!searched) {
                        TreeNode<K,V> q, ch;
                        searched = true;
                        if (((ch = p.left) != null &&
                             (q = ch.find(h, k, kc)) != null) ||
                            ((ch = p.right) != null &&
                             (q = ch.find(h, k, kc)) != null))
                            return q;
                    }
                    // 找不到，那么接下来会put值
                    dir = tieBreakOrder(k, pk);
                }

                TreeNode<K,V> xp = p;
                if ((p = (dir <= 0) ? p.left : p.right) == null) {
                // 这里该放新节点了
                    Node<K,V> xpn = xp.next;
                    TreeNode<K,V> x = map.newTreeNode(h, k, v, xpn);
                    if (dir <= 0)
                        xp.left = x;
                    else
                        xp.right = x;
                    xp.next = x;
                    x.parent = x.prev = xp;
                    // 维护双向链表
                    if (xpn != null)
                        ((TreeNode<K,V>)xpn).prev = x;
                    moveRootToFront(tab, balanceInsertion(root, x));
                    return null;
                }
            }
        }

        /**
         * Removes the given node, that must be present before this call.
         * This is messier than typical red-black deletion code because we
         * cannot swap the contents of an interior node with a leaf
         * successor that is pinned by "next" pointers that are accessible
         * independently during traversal. So instead we swap the tree
         * linkages. If the current tree appears to have too few nodes,
         * the bin is converted back to a plain bin. (The test triggers
         * somewhere between 2 and 6 nodes, depending on tree structure).
         */
        final void removeTreeNode(HashMap<K,V> map, Node<K,V>[] tab,
                                  boolean movable) {
            int n;
            if (tab == null || (n = tab.length) == 0) // 若表为null，直接返回
                return;
            int index = (n - 1) & hash; // 找到桶的索引
            TreeNode<K,V> first = (TreeNode<K,V>)tab[index], root = first, rl;
            TreeNode<K,V> succ = (TreeNode<K,V>)next, pred = prev;
            // 处理树的链表
            if (pred == null) // 如果第一个就是要删的节点，第一个节点变成下一个节点
                tab[index] = first = succ;
            else // 不是第一个的话，让前一个节点的下一个变成next
                pred.next = succ;
                // 整理下一个节点的link
            if (succ != null)
                succ.prev = pred;
            if (first == null)
                return;
           
           // 如果不是根节点，找到根节点
            if (root.parent != null)
                root = root.root();
            if (root == null || root.right == null ||
                (rl = root.left) == null || rl.left == null) {
                tab[index] = first.untreeify(map);  // 这些情况下，小于8个，需要退化成链表
                return;
            }
            
            TreeNode<K,V> p = this, pl = left, pr = right, replacement;
            if (pl != null && pr != null) { // 左右节点都不为null
                TreeNode<K,V> s = pr, sl;
                while ((sl = s.left) != null) // 找到继任者，其实就是红黑树中比该节点大的最小值，只是在这里无法中序遍历
                    s = sl;
                boolean c = s.red; s.red = p.red; p.red = c; // 当前节点和对调的节点交换颜色
                TreeNode<K,V> sr = s.right;
                TreeNode<K,V> pp = p.parent;
                if (s == pr) { // p是s的父亲，调换位置
                    p.parent = s;
                    s.right = p;
                }
                else { // 不是的话，也要对调
                    TreeNode<K,V> sp = s.parent;
                    if ((p.parent = sp) != null) {
                        if (s == sp.left)
                            sp.left = p;
                        else
                            sp.right = p;
                    }
                    if ((s.right = pr) != null)
                        pr.parent = s;
                }
                
                p.left = null;
                if ((p.right = sr) != null) // p的右边=s的右边，s的右边肯定比p的右边大
                    sr.parent = p;
                if ((s.left = pl) != null) // s的左边变成当前节点的左边
                    pl.parent = s;
                if ((s.parent = pp) == null) // 如果s是根节点，则root=s
                    root = s;
                else if (p == pp.left) // 如果pp的孩子，换成s----狸猫换太子
                    pp.left = s;
                else
                    pp.right = s;
                    
                if (sr != null) // 右边不等于null的话
                    replacement = sr;
                else // 右边为null，那么p就是最后的节点了
                    replacement = p;
            }
            else if (pl != null) // 左边等于null的话
                replacement = pl;
            else if (pr != null) // 右边等于null的话
                replacement = pr;
            else // 左右都为null的话
                replacement = p;
                
                // p不是最后一个节点的话，替换掉该节点
            if (replacement != p) {
                TreeNode<K,V> pp = replacement.parent = p.parent;
                if (pp == null)  // p就是root的话
                    root = replacement;
                else if (p == pp.left)
                    pp.left = replacement;
                else
                    pp.right = replacement;
                // 清除p的引用
                p.left = p.right = p.parent = null;
            }

// 如果p是黑色的，那么会影响平衡，得平衡节点
            TreeNode<K,V> r = p.red ? root : balanceDeletion(root, replacement);

            if (replacement == p) {  // 如果就是最后一节点，直接删除
                TreeNode<K,V> pp = p.parent;
                p.parent = null;
                if (pp != null) {
                    if (p == pp.left)
                        pp.left = null;
                    else if (p == pp.right)
                        pp.right = null;
                }
            }
            if (movable) // 只有迭代器在删除时候不能把根节点移到前面来，否则迭代会出错，本身这个就是为了性能考虑的
                moveRootToFront(tab, r);
        }

        /**
         * Splits nodes in a tree bin into lower and upper tree bins,
         * or untreeifies if now too small. Called only from resize;
         * see above discussion about split bits and indices.
         *
         * @param map 当前表
         * @param tab 新的表
         * @param index t当前桶索引
         * @param bit 旧桶数
         */
         
         // 这部分跟链表的拆分差不多
        final void split(HashMap<K,V> map, Node<K,V>[] tab, int index, int bit) {
            TreeNode<K,V> b = this;
            // Relink into lo and hi lists, 保持顺序
            TreeNode<K,V> loHead = null, loTail = null;
            TreeNode<K,V> hiHead = null, hiTail = null;
            int lc = 0, hc = 0;
            
            // 循环拆分
            for (TreeNode<K,V> e = b, next; e != null; e = next) {
                next = (TreeNode<K,V>)e.next;
                e.next = null;
                if ((e.hash & bit) == 0) {
                    if ((e.prev = loTail) == null)
                        loHead = e;
                    else
                        loTail.next = e;
                    loTail = e;
                    ++lc;
                }
                else {
                    if ((e.prev = hiTail) == null)
                        hiHead = e;
                    else
                        hiTail.next = e;
                    hiTail = e;
                    ++hc;
                }
            }

// 拆分后的整理
            if (loHead != null) {
                if (lc <= UNTREEIFY_THRESHOLD)
                    tab[index] = loHead.untreeify(map);
                else {
                    tab[index] = loHead;
                    if (hiHead != null) // (else is already treeified)
                        loHead.treeify(tab);
                }
            }
            if (hiHead != null) {
                if (hc <= UNTREEIFY_THRESHOLD)
                    tab[index + bit] = hiHead.untreeify(map);
                else {
                    tab[index + bit] = hiHead;
                    if (loHead != null)
                        hiHead.treeify(tab);
                }
            }
        }

        // Red-black tree methods, all adapted from CLR

// 与TreeMap一样，不重复注释
        static <K,V> TreeNode<K,V> rotateLeft(TreeNode<K,V> root,
                                              TreeNode<K,V> p) {
            TreeNode<K,V> r, pp, rl;
            if (p != null && (r = p.right) != null) {
                if ((rl = p.right = r.left) != null)
                    rl.parent = p;
                if ((pp = r.parent = p.parent) == null)
                    (root = r).red = false;
                else if (pp.left == p)
                    pp.left = r;
                else
                    pp.right = r;
                r.left = p;
                p.parent = r;
            }
            return root;
        }

// 与TreeMap一样，不重复注释
        static <K,V> TreeNode<K,V> rotateRight(TreeNode<K,V> root,
                                               TreeNode<K,V> p) {
            TreeNode<K,V> l, pp, lr;
            if (p != null && (l = p.left) != null) {
                if ((lr = p.left = l.right) != null)
                    lr.parent = p;
                if ((pp = l.parent = p.parent) == null)
                    (root = l).red = false;
                else if (pp.right == p)
                    pp.right = l;
                else
                    pp.left = l;
                l.right = p;
                p.parent = l;
            }
            return root;
        }

// 与TreeMap一样，不重复注释
        static <K,V> TreeNode<K,V> balanceInsertion(TreeNode<K,V> root,
                                                    TreeNode<K,V> x) {
            x.red = true;
            for (TreeNode<K,V> xp, xpp, xppl, xppr;;) {
                if ((xp = x.parent) == null) {
                    x.red = false;
                    return x;
                }
                else if (!xp.red || (xpp = xp.parent) == null)
                    return root;
                if (xp == (xppl = xpp.left)) {
                    if ((xppr = xpp.right) != null && xppr.red) {
                        xppr.red = false;
                        xp.red = false;
                        xpp.red = true;
                        x = xpp;
                    }
                    else {
                        if (x == xp.right) {
                            root = rotateLeft(root, x = xp);
                            xpp = (xp = x.parent) == null ? null : xp.parent;
                        }
                        if (xp != null) {
                            xp.red = false;
                            if (xpp != null) {
                                xpp.red = true;
                                root = rotateRight(root, xpp);
                            }
                        }
                    }
                }
                else {
                    if (xppl != null && xppl.red) {
                        xppl.red = false;
                        xp.red = false;
                        xpp.red = true;
                        x = xpp;
                    }
                    else {
                        if (x == xp.left) {
                            root = rotateRight(root, x = xp);
                            xpp = (xp = x.parent) == null ? null : xp.parent;
                        }
                        if (xp != null) {
                            xp.red = false;
                            if (xpp != null) {
                                xpp.red = true;
                                root = rotateLeft(root, xpp);
                            }
                        }
                    }
                }
            }
        }

// 与TreeMap一样，不重复注释
        static <K,V> TreeNode<K,V> balanceDeletion(TreeNode<K,V> root,
                                                   TreeNode<K,V> x) {
            for (TreeNode<K,V> xp, xpl, xpr;;)  {
                if (x == null || x == root)
                    return root;
                else if ((xp = x.parent) == null) {
                    x.red = false;
                    return x;
                }
                else if (x.red) {
                    x.red = false;
                    return root;
                }
                else if ((xpl = xp.left) == x) {
                    if ((xpr = xp.right) != null && xpr.red) {
                        xpr.red = false;
                        xp.red = true;
                        root = rotateLeft(root, xp);
                        xpr = (xp = x.parent) == null ? null : xp.right;
                    }
                    if (xpr == null)
                        x = xp;
                    else {
                        TreeNode<K,V> sl = xpr.left, sr = xpr.right;
                        if ((sr == null || !sr.red) &&
                            (sl == null || !sl.red)) {
                            xpr.red = true;
                            x = xp;
                        }
                        else {
                            if (sr == null || !sr.red) {
                                if (sl != null)
                                    sl.red = false;
                                xpr.red = true;
                                root = rotateRight(root, xpr);
                                xpr = (xp = x.parent) == null ?
                                    null : xp.right;
                            }
                            if (xpr != null) {
                                xpr.red = (xp == null) ? false : xp.red;
                                if ((sr = xpr.right) != null)
                                    sr.red = false;
                            }
                            if (xp != null) {
                                xp.red = false;
                                root = rotateLeft(root, xp);
                            }
                            x = root;
                        }
                    }
                }
                else { // symmetric
                    if (xpl != null && xpl.red) {
                        xpl.red = false;
                        xp.red = true;
                        root = rotateRight(root, xp);
                        xpl = (xp = x.parent) == null ? null : xp.left;
                    }
                    if (xpl == null)
                        x = xp;
                    else {
                        TreeNode<K,V> sl = xpl.left, sr = xpl.right;
                        if ((sl == null || !sl.red) &&
                            (sr == null || !sr.red)) {
                            xpl.red = true;
                            x = xp;
                        }
                        else {
                            if (sl == null || !sl.red) {
                                if (sr != null)
                                    sr.red = false;
                                xpl.red = true;
                                root = rotateLeft(root, xpl);
                                xpl = (xp = x.parent) == null ?
                                    null : xp.left;
                            }
                            if (xpl != null) {
                                xpl.red = (xp == null) ? false : xp.red;
                                if ((sl = xpl.left) != null)
                                    sl.red = false;
                            }
                            if (xp != null) {
                                xp.red = false;
                                root = rotateRight(root, xp);
                            }
                            x = root;
                        }
                    }
                }
            }
        }

        /**
         * 递归检查节点的合法性
         */
        static <K,V> boolean checkInvariants(TreeNode<K,V> t) {
            TreeNode<K,V> tp = t.parent, tl = t.left, tr = t.right,
                tb = t.prev, tn = (TreeNode<K,V>)t.next;
            if (tb != null && tb.next != t)
                return false;
            if (tn != null && tn.prev != t)
                return false;
            if (tp != null && t != tp.left && t != tp.right)
                return false;
            if (tl != null && (tl.parent != t || tl.hash > t.hash))
                return false;
            if (tr != null && (tr.parent != t || tr.hash < t.hash))
                return false;
            if (t.red && tl != null && tl.red && tr != null && tr.red)
                return false;
            if (tl != null && !checkInvariants(tl))
                return false;
            if (tr != null && !checkInvariants(tr))
                return false;
            return true;
        }
```
