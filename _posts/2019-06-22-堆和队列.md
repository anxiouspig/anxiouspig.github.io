# 优先队列

#### 什么是优先队列

普通队列：先进先出，后进后出

优先队列：出队顺序和入队顺序无关，和优先级相关

| 结构         | 入队    | s       |
| ------------ | ------- | ------- |
| 普通线性结构 | O(1)    | O(n)    |
| 顺序线性结构 | O(n)    | O(1)    |
| 堆           | O(logn) | O(logn) |
